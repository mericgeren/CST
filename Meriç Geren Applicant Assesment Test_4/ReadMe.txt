Bu proje, bir text dosyasında
betimlenmiş olan satranç tahtasındaki
siyah ve beyaz tarafların mevcut 
durumlarının puan hesaplamasını yapar.

Proje Java dilinde yazılmıştır. 

Derleme ve çalıştırma ortamı 
Eclipse Java 2019-09'dur. 

Bu yazılımı kullanmanız için önce, proje
klasörünü launch edeceğiniz workspace'e
koymalısınız. Eclipse'i proje klasörünün
olduğu workspace'de launch ettikten sonra
Project Explorer ile CST_Chess projesini 
bulup, src ve ardından default package'e 
tıklayıp, girmelisiniz. Package içinde 
CSTRunner'a girip, yukarıdaki panelde 
bulunan yeşil daire içinde beyaz üçgenin 
olduğu tuşa basarak programı
çalıştırabilirsiniz. Alttaki konsolda 
program sizden hesaplama yapılacak satranç
tahtasını girmenizi isteyecektir. konsolun
içine hesaplama yapılmasını istediğiniz 
tahtayı veya başka bir deyişle text 
dosyasyının adını girin. Bu durumda ".txt"
uzuntısını girmenize gerek yoktur. Program
sizin için uzantıyı ekleyecektir. Eğer,
girdiğiniz dosya bulunamazsa, konsola 
dosyanın bulunamadığına dair bir hata 
mesajı çıkacak ve ardından program 
sonlanacaktır. Eğer, dosya bulunursa, 
program hesaplamayı yapacak ve ardından 
siyahların puanını "Blacks:" olarak, 
beyazların puanını ise "Whites: " olarak
konsolda gösterecektir.
